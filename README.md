💻 Programming Class – C++ Learning Repository

📌 Introduction
This repository documents my progression in learning C++ programming through a series of structured exercises and practical implementations. It reflects both theoretical understanding and hands-on problem solving developed during my programming coursework.

The objective is not only to write code, but to understand core programming logic, build discipline in structuring solutions, and progressively move toward more advanced software development concepts.


🧠 Learning Scope

This repository covers fundamental programming concepts that form the backbone of computer science:

🔹 Core Concepts
- Variables, data types, and operators  
- Input and output handling  
- Control flow (if / else conditions)  
- Loop structures (for, while)  

🔹 Data Handling
- Arrays and basic data structures  
- Iteration and data processing  

🔹 Modularity
- Functions and code organization  
- Reusability and logical decomposition  

🔹 File Management
- File input/output operations  
- Writing and reading structured data  

---

📂 Repository Structure

The repository is organized as a collection of independent exercises, each focusing on a specific concept:

| File | Description |
|------|------------|
| `loop*.cpp` | Practice with loop structures and iterations |
| `array*.cpp` | Array manipulation and data storage |
| `function*.cpp` | Function definition and usage |
| `ifStatement*.cpp` | Conditional logic implementation |
| `whileStatement*.cpp` | While loop applications |
| `test.cpp` | General testing and experimentation |
| `io2.txt` | Output file used for file handling exercises |

Each file represents a step in the learning process, gradually increasing in complexity.
